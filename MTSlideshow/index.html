MTSlideshow

Gordon College, in Wenham, Massachusetts, is among the top Christian colleges in the nation and the only multidenominational Christian college in New England.

MTSlideshow is a simple javascript slideshow. It was built as an alternative for Gordon College's Flash-based home page slider. It has been implemented at USAi.net.

Because it is Javascript based, and does not require flash, MTSlideshow can be used to provide slider functionality that is compatible with Mobile WebKit Browsers.
